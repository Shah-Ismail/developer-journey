# 🧠 Developer Journey: Week 1 Reboot Notes

## 🖥️ C++: System-Level Foundations
* **Structs:** Blueprints for data. Used to group different types (int, string) together.
* **Functions & Scope:** Modular code blocks. In C++, passing by **reference** (`&`) is key to modifying data without extra memory cost.
* **Switch Statements:** Efficient multi-way branching. Faster than long `if-else` chains due to compiler jump tables.
* **File Handling (`fstream`):** * `ofstream` = Output to file.
    * `ifstream` = Input from file.
* **Compilation:** The 4-stage journey from code to binary (Preprocessing -> Compilation -> Assembly -> Linking).

## 🐍 Python: High-Level Scripting
* **Dynamic Lists:** Flexible arrays. `.append()` is O(1) complexity. 
* **Context Managers (`with`):** Automatically handles file closing. Prevents memory leaks and file corruption.
* **Exception Handling (`try...except`):** Prevents crashes from external errors (like missing files). Makes code "Robust."
* **List Comprehensions:** Concise one-line logic, e.g., `[line.strip() for line in file]`.
